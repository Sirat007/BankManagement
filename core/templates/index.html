Welcome to NovaBank

NovaBank is a simple banking system that I developed, offering core features such as depositing and withdrawing money, taking loans, and managing user accounts. Users can create an account, log in, perform transactions, and securely log out. The project is designed to provide a basic yet functional simulation of real banking operations, making it easy to understand and interact with. While the concept is straightforward, it effectively demonstrates key elements of user authentication, transaction handling, and system flow.
